# Інтерактивний Кулінарний Голосовий Бот

Цей проєкт реалізує голосового помічника для кулінарії (Варіант 14), який сприймає голосові команди, запам'ятовує інгредієнти, пропонує рецепти та веде покрокову інструкцію.

## Вимоги

* Python 3.9+
* Налаштовані API ключі для Groq, Gemini.
* Telegram Bot Token.

## Як запустити локальне тестування ШІ (без Telegram)

Це дозволяє перевірити взаємодію з Gemini та роботу TTS:

1. Відкрийте термінал у папці проєкту (`culinary_bot`).
2. Активуйте віртуальне середовище:
   ```bash
   .venv\Scripts\activate
   ```
3. Запустіть скрипт:
   ```bash
   python local_test.py
   ```
4. Спілкуйтеся з помічником через текст. Відповіді будуть озвучені та збережені у `response.mp3`.

## Як запустити Telegram Бота

1. Додайте ваш токен від BotFather у файл `.env` (значення `TG_BOT_TOKEN=...`).
2. Переконайтесь, що віртуальне середовище активовано (`.venv\Scripts\activate`).
3. Запустіть бота:
   ```bash
   python bot.py
   ```
4. Знайдіть свого бота в Telegram та натисніть `/start`! Ви можете надсилати йому як текстові, так і голосові повідомлення.
